# PC Repair Shop

A simple, responsive web application for a PC repair shop, built with React, Vite, and Tailwind CSS. The site features a premium "Dark Chrome Lab" aesthetic with dynamic animations, service listings, a contact form, and an interactive UI.

## Features
- **Dark Chrome Aesthetic**: Premium industrial tech vibe designed with Tailwind CSS.
- **Dynamic Starfield Background**: Interactive animated background using React.
- **Service Listings**: Clear display of offered repair services.
- **Responsive Design**: Works on mobile, tablet, and desktop devices.
- **Testing**: Comprehensive tests configured with Vitest and React Testing Library.
- **Containerized**: Docker and Docker Compose configured for standard deployments.
- **VPS Ready**: Includes Nginx configuration and a deployment guide for VPS hosting.

## Tech Stack
- Frontend: React 19, Vite, React Router DOM, Tailwind CSS V4
- Icons: React Icons
- Testing: Vitest
- Deployment: Docker, Docker Compose, Nginx
